# 📊 Sales Data Analysis Dashboard (Excel)

![Dashboard](PFD.png)

## 📌 Project Overview

This project analyzes sales data to uncover key business insights related to revenue, product performance, and regional trends. Using Microsoft Excel, the data was cleaned, processed, and transformed into an interactive dashboard to support data-driven decision-making.

## 🎯 Objectives

* Track overall sales performance
* Identify top-performing products and categories
* Analyze region-wise sales distribution
* Understand monthly sales trends
* Build an interactive dashboard for stakeholders

## 🛠️ Tools & Technologies

* Microsoft Excel

  * Pivot Tables
  * Pivot Charts
  * Slicers
  * Data Cleaning & Formatting
  * Functions (SUM, IF, etc.)

## 📊 Key Analysis Performed

* Total Sales and Revenue analysis
* Product Category performance
* Region-wise sales comparison
* Monthly sales trend analysis
* Order and customer-level insights
  
## 📈 Dashboard Features

* Interactive filters using slicers
* Dynamic charts for real-time insights
* KPI indicators (Total Sales, Orders, etc.)
* Clean and user-friendly layout

## 💡 Key Insights

* Identified top-performing products contributing to maximum revenue
* Found regions generating the highest sales
* Observed trends in monthly sales performance
* Highlighted underperforming areas for improvement

## 📂 Dataset

The dataset contains sales-related information such as:

* Order ID
* Product Category
* Sales Amount
* Region
* Date

## 🚀 Conclusion

This project demonstrates my ability to clean, analyze, and visualize data using Excel. It showcases how raw data can be transformed into meaningful insights through structured analysis and dashboard creation.
